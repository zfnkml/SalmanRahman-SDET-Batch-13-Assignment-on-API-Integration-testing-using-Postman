# About D-money project

A transaction System 

- Users
    1. Agent
    2. Customer
    3. Merchant

- Features
    1. Deposit Money
    2. Send Money
    3. Withdraw Money
    4. Payment

- Extra Features
    1. Account-Balance
    2. Account-Statement
    3. Search transaction

# 🖥️ Technology used:

- Postman
- Newman
- Node.js

# 🖥️ How to run?

- Clone this project
- PUT .env file
- Give following comand:
    
    ```jsx
    npm i
    npm test
    ```
